# UnseenConnect — Windows

| | |
|--|--|
| [💿 Скачать установщик](https://github.com/norachchan/unseenconnect-desktop/releases/latest/download/UnseenConnect-Setup.exe) | Последняя версия · x64 |

UnseenConnect — приложение для **Windows** для удобного управления прокси-серверами на базе ядра Xray. Простой интерфейс, работа из трея, импорт подписки и быстрое подключение к нужному серверу.

---

## Установка

1. Скачайте **UnseenConnect-Setup.exe** по кнопке выше.
2. Запустите установщик — Windows запросит права администратора (UAC).
3. Подтвердите и следуйте инструкциям установщика.
4. Вставьте ссылку на подписку или конфиг vless://.
5. Выберите сервер и нажмите **Подключить**.

> Требуется **Windows 10 x64** или новее.

---

## Возможности

* Импорт подписки и конфигов в один клик
* Системный прокси и полноценный VPN-режим (TUN)
* Список серверов с пингом и выбором лучшего узла
* Автообновление подписки и автоподключение
* Работа в фоне из системного трея
* Автозапуск вместе с Windows
* Автоматическое обновление приложения
* Тёмная тема, поддержка нескольких языков

## Поддерживаемые протоколы

* **VLESS (Reality)**
* **VMess**
* **Trojan**
* **Shadowsocks**
* **SOCKS**

---

UnseenConnect не собирает ваши данные — всё хранится только на вашем компьютере.

Приложение **не продаёт VPN-подписки**. Серверы вы подключаете самостоятельно у своего провайдера. Используйте сервис в соответствии с законодательством вашей страны.
