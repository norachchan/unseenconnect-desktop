UnseenConnect — приложение для **Windows** для удобного управления прокси-серверами. Простой интерфейс, работа из трея, импорт подписки и быстрое подключение к нужному серверу.

**Возможности:**

* Импорт подписки и конфигов в один клик
* Системный прокси и полноценный VPN-режим
* Список серверов с пингом и выбором лучшего узла
* Автообновление подписки и автоподключение
* Тёмная тема, несколько языков
* Работа из системного трея

**Поддерживаемые протоколы:**

* **VLESS (Reality)**
* **VMess**
* **Trojan**
* **Shadowsocks**
* **SOCKS**

UnseenConnect не собирает ваши данные — всё хранится только на вашем компьютере.

Приложение **не продаёт VPN-подписки**. Серверы вы подключаете самостоятельно у своего провайдера. Используйте сервис в соответствии с законодательством вашей страны.

## Как начать

1. Скачайте и установите **UnseenConnect-Setup.exe**.
2. Вставьте ссылку на подписку или конфиг.
3. Выберите сервер и нажмите **Подключить**.
